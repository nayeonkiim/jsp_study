< id 중복체크 누를 경우>
버튼 누른 경우 - js로 구현
- id 입력 안되어 있는 경우 : alert창으로 id 입력해주세요 메시지.
- id 입력 되어 있는 경우 : idCheck.jsp로 이동


<로그인 버튼 누를 경우>

1.member.jsp 에서 로그인 버튼 누르면 login.jsp 페이지로 이동,

회원가입에 성공하면 login.jsp 페이지로 이동

2. 아이디와 비밀번호를 입력하고 로그인 버튼을 누르면 아이디와 비밀번호 작성 유무 검사 후 모두 작성했으면

loginProc.jsp 페이지로 이동

3. loginProc.jsp에서 로그인 성공 유무를 알려주는 창이 뜨며

로그인 성공, 실패시 모두 login.jsp로 이동하여 상황에 맞는 메시지 출력됨.


﻿<회원가입 버튼 누를 시>


1.  각 항목(id,pw,이름,직업 등)이 모두 작성되었나 확인 : script.js의 inputCheck()함수 

 2. <form action="memberProc.jsp">을 통해 memberProc.jsp로 이동 후 

회원가입의 유무 알려줌

3. 회원가입 성공시 성공 메시지와 login.jsp로 실패시 실패 메시지와 member.jsp로 이동



﻿﻿<logout 버튼 누른 경우>

1. 버튼 누르면 logout.jsp로 이동 후 세션 종료하고 logout 되었다는 메시지 출력

login.jsp 페이지로 이동
